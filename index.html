My Projects

Pet Adoption

This comprehensive web application is designed to help users find pets available for adoption. It features a search functionality that allows users to look for pets based on type, breed, and location. The application is built with a Python backend using the Flask framework, and a frontend created with HTML, CSS. It integrates with a pet adoption API to provide the most current listings and includes a user authentication system to enhance user experience and functionality.

Bus Booking App

This full-stack web application is created to simplify online bus ticket booking for users. It includes functionalities such as searching for buses by route, date, and time, choosing seats, and processing payments. Utilizing the MERN stack—MongoDB for the database, Express.js and Node.js for the backend, and React.js for the frontend—the application provides a smooth user experience with real-time updates and secure transaction handling.
